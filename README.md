# Тестовый проект для пинга поисковых систем

## Описание

Этот проект содержит скрипт на Python, который пингует 5 известных поисковых систем в течение 10 секунд и выводит среднюю скорость ответа.

## Файлы

- `testFile.py` - основной скрипт для пинга поисковых систем
- `README.md` - документация проекта

## Поисковые системы

Скрипт пингует следующие поисковые системы:
1. Google (google.com)
2. Bing (bing.com)
3. Yahoo (yahoo.com)
4. DuckDuckGo (duckduckgo.com)
5. Yandex (yandex.com)

## Использование

Для запуска скрипта выполните команду:

```bash
python3 testFile.py
```

## Требования

- Утилита `ping` (доступна в Linux/macOS по умолчанию)
- Python 3.x

## Примечание

На Windows может потребоваться адаптация команды ping, так как синтаксис отличается от Unix-систем.
